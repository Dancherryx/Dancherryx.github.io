# Sweet Cherry

Welcome to the **Sweet Cherry** repository, a website created to promote our social media channels, delicious desserts (cookies, gelatins, standard cakes, custom cakes, petit fours, and tarts), and much more.

## Project Objective

The main goal of this Github Page is to serve as an attractive and functional digital showcase that allows our customers to discover our variety of products and easily connect with our social media.

## Project Structure

The project is organized into the following main sections for our desserts:

*   **Cookies**
*   **Gelatin**
*   **Standard Cakes**
*   **Custom Cakes**
*   **Petit Fours**
*   **Tarts**

## Development and Design

For the frontend development of this project, we use the artificial intelligence skill focused on design: `.agents\skills\frontend-design\SKILL.md`. This tool helps us create polished, creative, and high-quality user interfaces to highlight the presentation of our desserts.
